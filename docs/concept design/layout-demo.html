MES UI 프레임워크 레이아웃

레이아웃 개요

이 레이아웃은 MES(Manufacturing Execution System) 재개발을 위한 UI 프레임워크입니다. 멀티페이지 방식을 기반으로 하며, 데스크톱 우선이지만 모든 디바이스에서 반응형으로 동작합니다. 다크 테마를 지원하여 장시간 업무에 최적화되어 있습니다.

🔄 멀티페이지 아키텍처
각 탭이 독립적인 페이지로 동작하여 안정성을 확보하고, iframe 방식으로 화면 상태를 완벽하게 보존합니다.
🎨 다크 테마 지원
장시간 업무에 최적화된 다크 테마를 기본 제공하며, 라이트 테마로 전환도 가능합니다.
📱 반응형 디자인
데스크톱 우선이지만 태블릿과 모바일에서도 완벽하게 동작하는 반응형 레이아웃을 제공합니다.
🌳 트리 메뉴 시스템
기존 아코디언 메뉴를 5단계 트리 구조로 개선하여 더 유연하고 직관적인 네비게이션을 제공합니다.
📑 스마트 탭 관리
드래그앤드롭으로 탭 순서 변경 가능하며, 오버플로우 시 스크롤 처리로 많은 탭을 효율적으로 관리합니다.
⚡ 고성능 최적화
페이지 로딩 3초, 탭 전환 2초 이내의 성능 목표로 빠르고 쾌적한 사용자 경험을 제공합니다.

개발 일정

3개월 계획: 레이아웃(1주) → 헤더(2주) → 툴바(3-4주) → 메뉴(5-7주) → 탭(8-10주) → 반응형(11-12주)
